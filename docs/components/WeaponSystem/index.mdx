# Remarks

The Weapon System component is added to both the [BP_ThirdPersonCharacter](/classes/BP_ThirdPersonCharacter) and [BP_MasterWeapon](classes/BP_MasterWeapon) actors and splits the functionality between them. The [BP_ThirdPersonCharacter](/classes/BP_ThirdPersonCharacter) uses the weapon system to equip and unequip weapons as well as trigger the fire events. The [BP_MasterWeapon](/classes/BP_MasterWeapon) uses the to manage weapon data, and actually fire the weapon.

To initialize a weapon on the [BP_ThirdPersonCharacter](/classes/BP_ThirdPersonCharacter) ONLY, you need to select its pistol and/or rifle data:

---
title: AddWeaponUI(WeaponData:BP_WeaponData)
description: Adds Weapon UI to screen based on inserted Weapon Data parameter
---

# Actions

1. Calls [ClearWeaponUI()](/classes/BP_ThirdPersonCharacter/ClearWeaponUI) to clear the weapon UI currently on screen.
2. Get Weapon UI from [Weapon Data](/data/BP_WeaponData).
3. Sets Current Weapon UI variable.
4. Creates UI widget and adds it to the player's viewport.
4. Returns User Widget object reference.

---
title: StartRagdoll()
description: Activates ragdoll functionality for the player.
---

# Remarks

This function is usually called when the player dies. It should remove collision, visual meshes, spawn weapon pickups, and turn on the player's ragdoll.

---

# Actions

1. Disables Character Movement
2. Enables Ragdoll Physics on body mesh
3. Disables Capsule Component Collision
4. Hides Weapons
